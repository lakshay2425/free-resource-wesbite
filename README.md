<h1>Free Resources Website</h1>

It is a website where you can access free resources which are curated by me over a period of time. The resources are related to Tech, Productivity, Internship and Job Prepartion. You are welcome if you want to add your own resources as well.

<h3>Follow the instruction for contributing</h3>
1. Fork the repositary
2. Make another branch and make your changes.
3. Add your resources and commit the changes to your own repositary.
4. Follow similar format to add resources
5. Open a PR to make your changes accepted to the project.
6. Wait for the PR to get merged or changes you need to make.
